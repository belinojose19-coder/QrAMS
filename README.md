# QrAMS — Monorepo (web + PWA)

This repository contains a monorepo scaffold for the QrAMS attendance system.

Quick start:

```bash
# install pnpm (if you don't have it)
npm install -g pnpm

# install dependencies
pnpm install

# start development (runs Next.js web)
pnpm dev
```

The web app lives in [apps/web](apps/web).
